# PhonePe-pulse-data-visualization

## Project Overview
This project analyzes PhonePe Pulse transaction and user data using Power BI.  
The objective is to convert raw transaction and user data into interactive dashboards for business decision-making.

## Business Objective
The main objective is to identify:
- Transaction trends
- User growth
- Top revenue-generating districts
- Highest transaction states
- Payment category performance
- Regional digital payment adoption

## Tools Used
- Power BI
- DAX
- CSV Dataset
- Geo Visualization

## Dataset Used
The project uses six CSV files:
- aggregated_transaction.csv
- aggregated_user.csv
- map_transaction.csv
- map_user.csv
- top_transaction.csv
- top_user.csv

## Dashboards Created

### 1. Transaction Analysis Dashboard
Includes:
- Total Transaction Amount
- Total Transaction Count
- Total Users
- Average Transaction Value
- YoY Growth %
- QoQ Growth %
- Top 10 States by Transaction Value
- Payment Type Distribution
- State-wise Transaction Map

### 2. User Analysis Dashboard
Includes:
- Highest Revenue District
- Top Revenue District Amount
- Top 10 Revenue-Generating Districts
- State-wise Registered User Distribution

### 3. Geo Performance Dashboard
Includes:
- Regional Performance Summary
- Transaction Intensity by State
- State-wise digital payment performance

## Key Insights
- Merchant payments are the top transaction category.
- Some states contribute significantly higher transaction value.
- Bengaluru Urban district is the highest revenue-generating district.
- Geo visualization helps identify high-performing and low-performing regions.
- Power BI slicers make the dashboard interactive and user-friendly.

## Business Recommendations
1. Focus on high-performing states and districts to increase transaction volume.
2. Improve digital payment adoption in low-performing regions through offers and awareness campaigns.
3. Promote the top payment category through merchant partnerships and customer incentives.

## Conclusion
This project converts PhonePe Pulse data into meaningful business insights using Power BI.  
The dashboards help businesses track transactions, users, revenue districts, payment behavior, and regional performance across India.
